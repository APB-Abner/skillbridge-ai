
# SkillBridge.AI — Starter Kit (GS 2025 — Futuro do Trabalho)

Plataforma de requalificação com IA: diagnosticar gaps, recomendar trilhas (web + AR/VR), mentorias e analytics.

## Módulos
- `agile/` — visão, backlog, casos de uso e protótipo
- `database-design/` — DER conceitual e lógico (esqueleto SQL)
- `backend-java/` — domínio, serviços, DAOs (Maven)
- `ds/` — notebook de regressão (baseline)
- `ar-vr/` — roteiro e checklist de prints/vídeo
- `network/` — topologia Packet Tracer e landing
- `frontend/` — landing simples para evidência S4
- `docs/` — checklist de evidências por disciplina

## MVP (escopo)
1. Diagnóstico de Skills (questionário curto)
2. Recomendação de Trilhas (stub/JSON)
3. Mentorias & Agenda (fila/atendimentos)
4. Simulação AR/VR curta (3 cenas)
5. Dashboard de Progresso (cards + previsão)
